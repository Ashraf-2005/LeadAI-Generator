================================================================================
                      KEY DESIGN DECISIONS
================================================================================

✅ "Human Always in Loop"
   - AI drafts; humans send
   - No auto-send button (prevents costly mistakes)

✅ "No Hallucination"
   - Prompts forbid fabricating facts
   - Scoring grounded only in provided fields
   - JSON parsing with retry + fallback

✅ "Zero Setup MVP"
   - JSON file fallback (no MongoDB needed)
   - Demo credentials included
   - Sample data provided

✅ "Provider Agnostic"
   - LLM client designed for easy swaps
   - Change one file to switch providers

✅ "Single-Role MVP"
   - No complex RBAC
   - Clear upgrade path to multi-role
